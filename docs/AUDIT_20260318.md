# Полный аудит бота — 2026-03-18

## Почему крипта не торгует

**Причина:** `BREAKOUT_QUALITY_MIN_SCORE=0.52` в `.env` (строка 165).

В бэктесте quality score считается в идеальных условиях (нет реального спреда) →
100% сделок проходят фильтр. В лайве реальные условия (спред, chase, slippage) дают
score ниже 0.52 → бот логирует `BREAKOUT SKIP quality 0.38 < 0.52` и не входит.

**Фикс:** `BREAKOUT_QUALITY_MIN_SCORE=0.0` (отключить) или `0.30` (мягкий фильтр).

Динамический символьный фильтр (`build_symbol_filters.py`) работает —
бот автоматически перестраивает каждые 30 минут (`FILTERS_AUTO_BUILD=1`).

## DeepSeek AI менеджер (ПЛАН НА БУДУЩЕЕ)

**Идея:** API-прослойка между сигналом стратегии и исполнением.

**Архитектура:**
1. Стратегия генерирует сигнал (symbol, side, entry, sl, tp)
2. Перед исполнением: запрос к DeepSeek API с контекстом:
   - Текущий сигнал (side, entry, sl, tp, reason)
   - Состояние рынка (ATR, RSI, тренд, корреляция с BTC)
   - История последних 5 сделок на этом символе (win/loss/pnl)
   - Текущая загрузка портфеля (open positions, equity, drawdown)
3. DeepSeek отвечает: `{"action": "execute|skip|adjust", "confidence": 0.8, "adjust_sl": null, "adjust_tp": null}`
4. Если action=execute → исполняем, action=skip → пропускаем

**Стоимость:** DeepSeek ~$0.14/M input tokens, ~$0.28/M output tokens
При 50 сигналах/день × ~500 tokens = 25K tokens/день ≈ $0.004/день

**Когда реализовать:** После накопления 200+ лайв-сделок для обучения промпта.
